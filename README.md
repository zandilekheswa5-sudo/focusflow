# 🚀 FocusFlow

A modern productivity dashboard that helps users organize tasks, manage deadlines, track progress, and stay productive.

FocusFlow was built as a portfolio project to demonstrate frontend development skills using modern web technologies.

## 🌐 Live Demo

https://focusflow-nine-gules.vercel.app/

---

## ✨ Features

- 🔐 User Authentication (Supabase)
- ✅ Create Tasks
- ✏️ Edit Tasks
- 🗑 Delete Tasks
- 📅 Due Dates
- 🔍 Search Tasks
- 🎯 Filter Tasks
- 🔴 High, Medium & Low Priority
- 📂 Categories
- 📊 Statistics Dashboard
- 📈 Progress Tracking
- 🔥 Daily Streak Counter
- 📅 Upcoming Deadlines
- 📱 Responsive Mobile & Desktop Design
- 🌙 Modern Dark UI
- 🎉 Confetti Celebration

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/zandilekheswa5-sudo/focusflow.git
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
